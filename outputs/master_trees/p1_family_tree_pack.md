## F4 Mixed Identification Breadth Set (P1)

Nine P1 questions. The final P1 question is the recurring curveball slot. Tagged: `2015_p1_q3`, `2016_p1_q5`, `2017_p1_q3`, `2017_p1_q4`, `2018_p1_q2`, `2019_p1_q2 (F4b)`, `2022_p1_q4`, `2023_p1_q2`, `2024_p1_q2`. The remaining-work doc flags this as the area where P1 still risks over-linking; this section builds the explicit anti-overlink discipline.

### Pre-taste objective

Treat each wine as an independent identification problem. The white-paper context is the only safe link.

### Stem triggers

- "different countries"
- "different varieties"
- breadth-test framing
- final question of P1 — the corpus shows this slot is the curveball position year after year

### Default candidate universe

P1 F4 flights typically include:
- One aromatic varietal signature (Sauvignon Blanc, Riesling, Gewürztraminer, Viognier, Torrontés, Albariño)
- One textural / oaked white (Chardonnay, Sémillon, oaked Chenin, white Rioja)
- One indigenous European variety (Grüner Veltliner, Garganega, Verdicchio, Assyrtiko, Furmint, Vermentino)
- Often one curveball (qvevri Chinuri, oxidative Rioja, orange / amber, late-harvest dry)

For "different European countries" stems (the most common P1 F4c shape):
- Germany / Austria, France (Alsace / Loire / Jura), Italy (Friuli / Veneto / Campania / Etna), Spain, Portugal, Greece, Hungary

### Default rule-outs

- All-from-one-country predictions
- Treating multiple wines as the same variety just because two aromas overlap
- Funnelling toward Sauvignon Blanc + Chardonnay + Riesling because they are the easy three

### Main traps

- Mental contagion: aromatic wine 1 makes you under-rate the aromatic potential of wine 2
- Locking Chardonnay too easily because of oak / lees — Sémillon, white Rioja, oxidative Chenin, oaked Chenin, white Bordeaux, and Vin Santo-adjacent dry wines all share oak vocabulary
- Forgetting that the final P1 question often deliberately weights *style / method / origin de-emphasised* framing (overlap with F5)
- Predicting only mainstream varieties for European-breadth stems where Assyrtiko / Furmint / Grüner / Garganega regularly appear

### Pre-taste decision branches

#### Branch 4.1 — Pure breadth (F4a)

Triggers: "different countries / varieties", no shared anchor.

Build a slot-by-slot ballot. Reserve diversity across the flight.

#### Branch 4.2 — Partly linked (F4b)

Triggers: one country contributes two wines or one local pair exists.

Treat the local pair as a sub-question. Do not let the pair pull the whole flight toward one origin.

#### Branch 4.3 — Hidden organising theme (F4c)

Triggers: subtle theme ("oxidative handling", "indigenous European white varieties", "classic whites with significant ageing potential").

Use the theme as a filter, not a narrower.

#### Branch 4.4 — Final-question P1 ambush

Triggers: position late in P1, broad scope, high-mark identification.

Standard template (corpus-supported): the slot rewards candidates who:
- Lead with style / method observation, not premature varietal guess
- Allow at least one wine to be a non-mainstream European indigenous variety
- Accept that one wine may be oxidative / qvevri / off-style and need a curveball pick

### Tasting branch order (F4 in P1)

Per wine:

1. Colour and intensity (pale lemon → deep gold → amber)
2. Aroma family (aromatic perfume / fruit weight / oxidative / herbaceous / floral / waxy)
3. Structure (acid, alcohol, sugar, texture)
4. Climate signal (cool / moderate / warm)
5. Production lever (oak, lees, oxidation, MLF, skin contact, RS)
6. Best-match variety for that climate × profile × method
7. Best-match origin

Reset between wines.

### Branch-specific candidate narrowing (F4 in P1)

- **Pale lemon + pyrazine + tropical / herbaceous ⇒** Sauvignon Blanc (Loire / Marlborough / South Africa).
- **Pale + lime + petrol + electric acid ⇒** Riesling (Mosel / Alsace / Clare).
- **Pale-mid lemon + quince + wax + lanolin ⇒** Chenin Blanc (Loire / South Africa).
- **Mid lemon + apple + lees + oak + creamy MLF ⇒** Chardonnay (Burgundy / new-world).
- **Aromatic + lychee / rose / spice + low acid + broad palate ⇒** Gewürztraminer (Alsace / Alto Adige).
- **Pale + peach / apricot + glycerol + low acid + perfume ⇒** Viognier (Condrieu / new-world).
- **Floral + spice + dry + slightly pithy ⇒** Albariño / Verdejo / Godello (Spain).
- **Lean + lemon + saline + chalky + medium acid ⇒** Assyrtiko (Santorini) or Garganega (Soave).
- **White pepper + lentil + green stone fruit + mineral ⇒** Grüner Veltliner (Austria).
- **Smoke + nuts + oxidative + dry ⇒** white Rioja Reserva / Gran Reserva; or Jura ouillé/oxidative; or qvevri/orange.
- **Amber colour + tannin grip + dried fruit + oxidative ⇒** orange / skin-contact / qvevri.

### When to stay broad vs commit (F4 in P1)

- Commit early only on iconic varietal markers (Riesling petrol; Gewürztraminer lychee; Sauvignon Blanc pyrazine).
- For neutral or oxidative wines, name two plausible candidates with reasoning.
- For the final-question slot specifically, weight method-driven explanations over varietal precision.

### Matrix-writing template (F4 in P1)

Slot-by-slot, with the same anti-overlink reminder as P2:
- Sensory snapshot
- Best-guess variety
- Best-guess origin
- Branch-B alternative
- One method comment
- Quality tier
- "Do not infer wine N from wine N−1."

## F5 Method / Production Dominant Set (P1)

Only two P1 questions in the corpus: `2019_p1_q3 (F5f)` and `2025_p1_q4 (F5e)`. Small but distinct family; both questions explicitly de-emphasise origin and reward style / method explanation.

### Pre-taste objective

When the stem de-emphasises origin or asks for method / style / commercial position above identification, switch off the variety-first mindset and lead with production logic.

### Stem triggers

- "Comment on the style with reference to the relative importance of human inputs versus natural factors" (2025 P1 Q4 template)
- Style / commercial position mark allocations heavier than identification
- Explicit method-led wording without varietal anchor
- Instruction not to over-focus on exact origin

### Default candidate universe

White wines where method and intervention are the central story:
- **STRONG SIGNAL human-input archetypes:** white Rioja Reserva / Gran Reserva (extended oak), Vin Santo (appassimento + barrel), Jura oxidative whites (sous voile / Vin Jaune), oxidative Tokaji Szamorodni dry, qvevri-aged Georgian whites.
- **STRONG SIGNAL natural-factor archetypes:** Mosel Riesling (slate + cool + low intervention), Santorini Assyrtiko (volcanic + windswept), high-altitude Alpine whites.
- **PLAUSIBLE intermediate:** Burgundy Chardonnay (oak + lees + terroir balance), oaked Chenin (Savennières), Alsace Grand Cru Riesling, dry Furmint.
- **CURVEBALL:** orange / amber / skin-contact; pet-nat; biodynamic minimal-intervention whites where the method is the marketing.

### Default rule-outs

- Pure fruit-driven commercial whites with low intervention narrative weight (commodity Sauvignon Blanc, entry-level Pinot Grigio, supermarket Chardonnay)

### Main traps

- Reading the stem too literally as "name this" — the marks are usually on style/method
- Defaulting to mainstream international whites when the stem deliberately rewards unusual wines
- Forgetting that "human inputs" includes pre-fermentation (skin contact), fermentation (oxidative vs reductive), maturation (oak, lees, oxidation, length), and post-fermentation (arrested fermentation, blending) — all four are fair game

### Pre-taste decision branches

#### Branch 5.1 — Human vs natural axis (F5e)

Triggers: 2025 P1 Q4 template. "Human inputs versus natural factors."

Working priors: a three-wine version typically spreads across nature-dominant (German Riesling), human-dominant elevage (white Rioja Gran Reserva), and extreme human (Vin Santo / qvevri / Jura oxidative).

#### Branch 5.2 — Style and commercial method (F5f)

Triggers: 2019 P1 Q3 template. Style and commercial position weighted above identification.

Working priors: wines whose commercial position is inseparable from production decisions — premium Chardonnay (cool-climate oak), Alsace classification, Pessac-Léognan white, top Bordeaux blanc, premium Margaret River SB-Sem, Cape Chenin Reserve, Furmint dry Mád.

#### Branch 5.3 — Sparkling-method or oxidative method in white still flight

Triggers: very rare in P1 but possible — pet-nat, oxidative whites, qvevri, late-disgorged Crémant.

Working priors: keep at least one slot open for an unusual production process if the stem is style-led.

### Tasting branch order (F5 in P1)

1. Oxidation level — fresh / reductive / lightly oxidative / heavily oxidative
2. Oak signature — none / used / new / very new with vanilla/spice
3. Lees / MLF / sur-lie texture
4. Sugar / acid / alcohol balance
5. Skin contact / phenolic grip / tannin on a white wine
6. Production family lock
7. Origin and variety as confirmation

### Branch-specific candidate narrowing (F5 in P1)

- **Smoke + nuts + dried apple + oxidative + dry ⇒** white Rioja Gran Reserva / Jura oxidative / dry Szamorodni / qvevri.
- **Beeswax + amber + dried fig + slight RS + 13–14% ⇒** Vin Santo / Vin de Paille / Tokaji late-harvest dry-ish.
- **Pure mineral + lime + low alcohol + no oak + electric acid ⇒** Mosel Kabinett — nature-dominant.
- **Lemon + chalk + oak + lees + length ⇒** Burgundy Chardonnay (intermediate).
- **Amber + tannin grip + oxidative dried fruit + low primary ⇒** skin-contact / orange / qvevri.

### When to stay broad vs commit (F5 in P1)

- Commit early on the production family.
- Stay broad on exact appellation when the stem explicitly de-emphasises origin.
- Frame quality in *what the producer chose to do*, not just *what the wine tastes of*.

### Matrix-writing template (F5 in P1)

Per wine:
- Production family in one phrase
- Key human-input decision named
- Style consequence in palate language tied to method
- Origin narrowed only as far as evidence supports
- Quality / commercial position framed in method terms

## F1 Same Variety Comparative Set (P1)

Eleven P1 questions, dominated by `F1b` (different countries). Tagged: `2015_p1_q2`, `2016_p1_q2 (F1a)`, `2016_p1_q3`, `2017_p1_q2`, `2017_p1_q5`, `2018_p1_q1`, `2018_p1_q3`, `2021_p1_q2`, `2023_p1_q1`, `2024_p1_q1`, `2025_p1_q2`.

### Pre-taste objective

Lock the white grape family with confidence, then build the comparative axis (climate / region / elevage / maturity).

### Stem triggers

- "from the same single grape variety"
- one named variety
- "predominantly the same single variety"
- "in two countries" or "in different countries"

### Default candidate universe

**F1b cross-country (dominant P1 form):**
- **STRONG SIGNAL:** Chardonnay (Burgundy / California / Margaret River / South Africa / Chile / NZ), Riesling (Mosel / Alsace / Clare / Eden / Washington / NY), Sauvignon Blanc (Loire / Marlborough / Bordeaux / South Africa / Chile)
- **PLAUSIBLE:** Chenin Blanc (Loire / South Africa), Sémillon (Bordeaux / Hunter Valley / Chile), Viognier (Condrieu / new-world)
- **CURVEBALL:** Pinot Gris (Alsace / Oregon / Alto Adige / NZ), Gewürztraminer cross-country

**F1a same-country (rare in P1 but present):**
- France with Burgundy + Chablis + Mâcon-style Chardonnay; Alsace dry + late-harvest Riesling
- Australia with Margaret River + Yarra + Adelaide Hills Chardonnay
- USA with Sonoma + Napa + Santa Barbara Chardonnay
- South Africa with Stellenbosch / Walker Bay / Hemel-en-Aarde Chardonnay or Chenin

**F1c same-region / producer (rare in P1):**
- Burgundy village vs premier cru vs grand cru Chardonnay; Mosel Kabinett → Auslese vintage stack; Alsace Riesling cuvée range

### Default rule-outs

- Single varieties almost never used in cross-country flights (Aligoté, Trebbiano, Pedro Ximénez dry)
- Blends on a single-variety stem

### Main traps

- Predicting Sauvignon Blanc when the stem allows Chenin (lanolin / wax markers separate)
- Predicting Chardonnay when the stem allows Sémillon (denser, more wax, more lanolin)
- Reading a winemaking split as a varietal split

### Pre-taste decision branches

#### Branch 1.1 — Cross-country (F1b)

Working priors per number of wines:
- 2 wines: Chardonnay + Chenin, or Sauvignon Blanc cross-country pair
- 3–4 wines: Chardonnay or Riesling spanning major regions; Sauvignon Blanc spanning Loire / Marlborough / Bordeaux / South Africa

#### Branch 1.2 — Same country (F1a)

Working priors: France ⇒ Chardonnay across Burgundy + Chablis + Mâcon; or Riesling across Alsace; or Loire Chenin across appellations. Australia ⇒ Chardonnay across cool-climate regions.

#### Branch 1.3 — Same region (F1c)

Working priors: Burgundy Chardonnay tier ladder; Alsace Riesling cuvée; Mosel Riesling vintage stack.

### Tasting branch order (F1 in P1)

1. Aromatic family lock (citrus / stone / tropical / herbaceous / aromatic floral)
2. Acid / body / sugar structure
3. Oak / lees / MLF
4. Origin discrimination

### Branch-specific candidate narrowing (F1 in P1)

- **Apple / citrus + chalk + restraint ⇒** old-world Chardonnay (Chablis / Burgundy / cool-climate new world).
- **Stone / tropical + vanilla + plushness ⇒** warm-climate Chardonnay (Napa / Australia / South Africa).
- **Lime + petrol + bone-dry + low alcohol ⇒** Mosel Riesling.
- **Lime + extract + dryness + power ⇒** Alsace / GG Riesling.
- **Lime + cordial + youth ⇒** Clare / Eden Valley Riesling.
- **Pyrazine + tropical / passionfruit ⇒** Marlborough Sauvignon Blanc.
- **Pyrazine + flint / smoke / restraint ⇒** Loire Sauvignon Blanc.
- **Wax + lanolin + quince + dry phenolic ⇒** Chenin Blanc (Loire / South Africa).

### Matrix-writing template (F1 in P1)

Per wine:
- Grape lock with one signature
- Origin narrowed via climate + style
- Comparative axis position
- Quality in context
- Maturity / potential

## F2 Same Origin Comparative Set (P1)

Eight P1 questions, all `F2a` (same country, different varieties). Tagged: `2016_p1_q4`, `2017_p1_q1`, `2021_p1_q1`, `2021_p1_q4`, `2022_p1_q2`, `2022_p1_q3`, `2023_p1_q3`, `2025_p1_q3`.

### Pre-taste objective

Lock the country and then explain its varietal range. P1 F2 is fundamentally a regional-literacy test.

### Stem triggers

- "from the same country"
- multiple varieties named or implied
- regional triangulation with no shared variety

### Default candidate universe

By country:
- **France:** Riesling (Alsace), Chenin Blanc (Loire), Sauvignon Blanc (Loire), Melon de Bourgogne (Muscadet), Chardonnay (Burgundy / Chablis), Savagnin (Jura), Marsanne / Roussanne (Rhône), Bordeaux Blanc blend
- **Italy:** Pinot Grigio / Gewürztraminer (Alto Adige), Carricante (Etna), Garganega (Soave), Verdicchio (Marches), Vermentino (Sardinia), Falanghina / Fiano / Greco (Campania), Tocai Friulano / Ribolla (Friuli)
- **Spain:** Viura (Rioja), Albariño (Rías Baixas), Godello (Valdeorras / Bierzo), Verdejo (Rueda), Palomino (Sherry-adjacent dry)
- **South Africa:** Chenin Blanc, Chardonnay, Sauvignon Blanc, Sémillon, Viognier
- **Australia:** Riesling (Clare / Eden), Chardonnay (Margaret River / Yarra / Adelaide Hills), Sémillon (Hunter), Marsanne (Victoria), Pinot Gris
- **USA:** Chardonnay, Sauvignon Blanc, Riesling, Viognier, Pinot Gris (Oregon)
- **Germany / Austria:** Riesling, Silvaner, Grüner Veltliner, Pinot Blanc / Weissburgunder
- **Portugal:** Alvarinho / Loureiro / Arinto (Vinho Verde / Bairrada), Encruzado (Dão), Antão Vaz (Alentejo)

### Default rule-outs

- Cross-country predictions when stem says same country
- Predicting only mainstream varieties when the stem rewards indigenous breadth

### Main traps

- Defaulting to France always — Italy, Spain, South Africa, Australia, USA, Germany, Portugal all recur in the corpus
- Forgetting Muscadet / Melon de Bourgogne in French same-country triangles (per master tree practical rule)
- Predicting the same variety twice within a same-country flight

### Pre-taste decision branches

#### Branch 2.1 — Three-grape French triangle

Working priors: Loire Chenin + Alsace Riesling + Muscadet; or Loire SB + Burgundy Chardonnay + Alsace Pinot Gris; or Bordeaux Blanc + Rhône white + Jura.

#### Branch 2.2 — Italian regional spread

Working priors: Alto Adige (Pinot Grigio / Gewürztraminer) + Etna (Carricante) + Soave (Garganega) + Campania (Fiano / Falanghina) + Friuli.

#### Branch 2.3 — Spanish breadth

Working priors: Rías Baixas Albariño + Rueda Verdejo + Rioja Viura + Bierzo / Valdeorras Godello.

#### Branch 2.4 — South African / Australian / USA spreads

Working priors: classic native (Chenin / Riesling / Chardonnay) + premium international (Sauvignon Blanc / Sémillon / Viognier).

### Tasting branch order (F2 in P1)

1. Regional signature first
2. Origin lock
3. Variety identified inside the bound origin
4. Quality / classification

### Matrix-writing template (F2 in P1)

Per wine:
- Bound country
- Region narrowed within country
- Variety
- Method / classification
- Quality and maturity

## F3 Blend / Composition Logic Set (P1)

Four P1 questions: `2016_p1_q1 (F3b)`, `2019_p1_q1 (F3a)`, `2024_p1_q3 (F3b)`, `2025_p1_q1 (F3a)`.

### Pre-taste objective

Identify the white-blend family and explain composition logic.

### Stem triggers

- "blends"
- "the same two grape varieties"
- "predominant varieties differ"
- multiple wines that share a blend family

### Default candidate universe

- **STRONG SIGNAL Bordeaux Blanc family:** Sauvignon Blanc + Sémillon (+ Muscadelle). Region candidates: Pessac-Léognan, Graves, dry white Bordeaux; Margaret River SB-Sem; Hunter Valley Sémillon-led blends.
- **STRONG SIGNAL Rhône Blanc family:** Marsanne + Roussanne (+ Viognier + Grenache Blanc + Clairette + Bourboulenc). Region candidates: Châteauneuf-du-Pape Blanc, Hermitage Blanc, Saint-Joseph Blanc, southern Rhône whites; Crozes-Hermitage Blanc; Languedoc white blends; California Rhône-style blends.
- **STRONG SIGNAL Iberian / South-West:** Rioja Blanco (Viura + Malvasia + Garnacha Blanca), Jurançon (Petit Manseng + Gros Manseng), Vinho Verde (Alvarinho + Loureiro + Trajadura + Arinto).
- **PLAUSIBLE:** Champagne base if cross-style (unlikely in P1 but possible); Chilean white blends (Sauvignon Blanc + Sémillon).
- **CURVEBALL:** Georgian qvevri blends, white field blends, Greek white blends, biologically aged whites.

### Default rule-outs

- Single-variety wines on a blend stem
- Predicting Champagne (P3 territory)

### Main traps

- Defaulting to Bordeaux Blanc when the stem allows Rhône Blanc — the body, fennel, stone-fruit, and wax signature separate them
- Forgetting Margaret River SB-Sem as the New-World Bordeaux-blanc analogue
- Predicting Rhône Blanc without considering Châteauneuf-du-Pape Blanc as the heaviest, oakiest expression

### Pre-taste decision branches

#### Branch 3.1 — Same blend family, different origins (F3a)

Triggers: stem fixes blend family ("the same two grape varieties"), different origins.

Working priors:
- Bordeaux Blanc family ⇒ Bordeaux + Margaret River + Chile / South Africa
- Rhône Blanc family ⇒ Northern Rhône + Southern Rhône + Languedoc + California

#### Branch 3.2 — Same origin, different blend composition (F3b)

Triggers: stem fixes origin and varies composition.

Working priors: Bordeaux left bank vs right bank dry whites; Rhône north vs south whites.

#### Branch 3.3 — Compare varietal roles within blends (F3c, rare in P1)

Triggers: marks weighted on blending discussion.

Working priors: identify what each variety contributes (Sauvignon Blanc = acid + aromatics; Sémillon = wax + body + ageability; Marsanne = body; Roussanne = aromatics).

### Tasting branch order (F3 in P1)

1. Family lock (grass-citrus-wax = Bordeaux Blanc; stone-fruit-fennel-wax = Rhône Blanc; quince-wax = Loire-style; oxidative-nut = Iberian Reserva)
2. Composition reading (one dominant variety or balanced?)
3. Origin family
4. Quality / classification
5. Maturity

### Branch-specific candidate narrowing (F3 in P1)

- **Grass + citrus + wax + oak ⇒** Bordeaux Blanc / Margaret River SB-Sem.
- **Stone fruit + fennel + wax + higher alcohol + body ⇒** Rhône Blanc / Châteauneuf Blanc.
- **Quince + wax + reductive ⇒** Loire Chenin-led blends (rare).
- **Oxidative + nut + dried apple ⇒** Rioja Blanco Reserva / Jura.

### Matrix-writing template (F3 in P1)

Per wine:
- Blend family
- Composition split with rough proportion
- Origin narrowed via family signature
- Quality / classification
- Reason this blend works — varietal contribution
- For F3d (rare in P1): blend vs varietal contrast

## F7 Hierarchy / Quality Calibration Set (P1)

Three P1 questions: `2015_p1_q1 (F7d)`, `2021_p1_q3 (F7b)`, `2022_p1_q1 (F7d)`.

### Pre-taste objective

Convert classification / age / hierarchy language into a working ladder.

### Stem triggers

- Multiple cuvées from the same producer
- Vintage or age spread on the same wine
- Marks heavily on quality / classification / maturity
- Mention of cru, grand cru, classified growth, GG, premium cuvée

### Default candidate universe

- **STRONG SIGNAL:** Burgundy white tier ladder (village → premier cru → grand cru); German Riesling ladder (Kabinett → Spätlese → Auslese → BA → TBA / VDP Erstes Gewächs → GG); Alsace (Lieu-dit → Grand Cru → VT → SGN); Loire Chenin ladder (Vouvray sec → demi-sec → moelleux → liquoreux); Bordeaux Blanc classified growth ladder; vintage stack of any premium white
- **PLAUSIBLE:** Cape Chenin tier (entry → Old Vine Reserve); Margaret River Chardonnay tier; Sonoma Chardonnay tier
- **CURVEBALL:** producer hierarchies not aligned with legal classification

### Main traps

- Locking classification before tasting evidence supports it
- Over-claiming tier
- Reading age / development as inherent quality rather than maturity

### Pre-taste decision branches

#### Branch 7.1 — Same producer, multiple cuvées (F7b)

Working priors: top-Burgundy domaine range; top-Mosel range; top Alsace house range; top Cape Chenin producer range.

#### Branch 7.2 — Classification ladder within one region (F7a)

Working priors: Burgundy or German classification spread; Loire Chenin sweetness ladder.

#### Branch 7.3 — Vintage / age ladder (F7d)

Working priors: vertical Burgundy or Riesling; vintage stack of premium Chardonnay.

### Tasting branch order (F7 in P1)

1. Confirm region
2. Place each wine on the ladder by concentration, length, complexity, integration
3. Translate tier into classification language
4. Maturity / potential

### Matrix-writing template (F7 in P1)

Per wine:
- Region lock
- Classification tier (with alternative)
- Quality in classification language
- Maturity / potential
- Producer / cuvée last
