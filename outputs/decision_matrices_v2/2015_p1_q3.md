## Anti-collapse note

**F4 anti-overlink discipline:** This is a pure breadth question. Each of the six wines is an independent identification problem. The IMW deliberately selected six different varieties across multiple countries and aromatic profiles. There is no shared thread to exploit. The only safe pre-taste link is that varieties used earlier in the paper (Chardonnay, Chenin Blanc) will not appear here. Do not assume a shared winemaking technique, vintage character, or regional style connects any two wines. Taste each wine as if it is the only wine in front of you.
